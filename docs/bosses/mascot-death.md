# 魂喰いの死神 (mascot-death)

## 基本情報

- **名前** / **個体名**: 魂喰いの死神（個体名は不明）
- **種族**: 死神（マスコット型の怪異）
- **生息地**: 天空地方の外れにある「黄昏の霊園」
- **外見**: ずんぐりした小さな黒ローブの死神マスコット。身の丈以上の大鎌を持つ
- **体格**: 小柄（大鎌の方が体より大きい）
- **体色**: 漆黒のローブ、フードの奥は真っ白でまんまるな顔

## コンセプト

「可愛らしいが恐ろしい」を体現した死神マスコット。闇魔法で相手を弱らせてから、1ターンの溜めを要する大鎌の一閃で魂ごと刈り取る。刈り取った魂は丸呑みにして体内に閉じ込め、生気を吸収し続ける。

## 設定・背景

黄昏の霊園にいつからか住み着いた、小さな死神のような怪異。本来の死神の務めである「魂のお迎え」を真似ているが、気に入った魂を見つけると刈り取ってそのまま自分のものにしてしまう。本人（？）に悪意はなく、刈り取った魂を「たからもの」として大切に扱うが、魂にとっては迷惑極まりない。頭の蝋燭の灯は、これまで吸収した生気の輝きだと言われている。

## 身体的特徴

- フードの奥にまんまるで真っ白な顔と、黒くつぶらな瞳
- 頭にちょこんと小さな蝋燭が灯っている。生気を吸うと灯が大きく燃え上がる
- 身の丈以上の大鎌。刃は月光のように淡く輝き、体を傷つけず魂だけを刈り取る
- 体内はほんのり温かい漆黒の空間で、奥に魂を乗せる小さな燭台がある

## 性格

- 無邪気で人懐っこいが、やることは恐ろしい
- きれいな魂への執着が強く、気に入ると「ずっといっしょ」にしようとする
- 悪意はゼロ。刈り取りも丸呑みも、本人にとっては宝物集めの延長

## 口調

- 一人称: ぼく / 二人称: きみ。ひらがな多めの舌足らずな話し方
- 「きみのたましい、とってもきれいだね...」
- 「かりとっちゃうよ...？」
- 「おなかのなか、あったかいでしょ？」

## 能力・戦術

- **通常時**: 鎌の柄での小突きや鬼火で牽制しつつ、「闇の魔弾」で魂の呪い（弱体化）を付与する。時々「影の縛り」で拘束を狙う
- **大鎌の一閃**: 魂の呪いで弱った相手に「大鎌振り上げ」（1ターン溜め）を行い、次のターンに「魂刈りの一閃」を放つ。防御しなければ一撃でダウンし、魂が体から抜け出た「魂状態」になる。防御すれば大ダメージで耐えられる
- **拘束中**: 影の手で縛った相手に闇魔法を流し込みつつ、3ターンの猶予ののち「拘束刈り取り」で魂を刈り取る
- **ダウン中**: 「魂の切り離し」で魂状態にし、抜け出た魂を「魂の丸呑み」で体内に閉じ込める
- **体内**: 魂から生気をちゅうちゅうと吸収する（HP吸収あり）。体内から脱出すれば、魂はすぐに元の体に戻る
- **とどめ**: 体内で力尽きた魂を燭台に乗せ、「永遠の魂の蝋燭」として永続的に生気を吸い続ける
- **敗北後**: 永遠の生気吸収・蝋燭眺め・魂磨き・子守唄。8ターンごとに「魔力の与え直し」で魂の灯に魔力を注ぎ直し、また吸い続ける

## 捕食方法

- 大鎌で体から刈り取った魂を、小さな両手で掬い上げてあむっと丸呑みにする
- 体そのものは食べない。あくまで魂だけを体内に閉じ込める

## 体内表現

- ほんのり温かい漆黒の空間。魂を優しく包み、生気を吸い上げる
- 奥には小さな燭台があり、力尽きた魂はそこに乗せられて蝋燭の灯になる

## 特殊状態異常

- **魂の呪い (SoulCurse)**: 闇魔法で魂と体の結びつきが緩む。攻撃力 0.8 倍・命中 0.9 倍、4ターン
- **死神の構え (ScytheStance)**: ボス自身に付く溜め状態の表示。次ターンの一閃の予告
- **魂状態 (SoulForm)**: 魂が体から刈り取られた状態。攻撃力 0.7 倍・もがき成功率 0.8 倍。体外で（ダウンしていない状態で）ボスのターンが来ると、魂はすぐに元の体へ戻る

## 主なギミック

### 大鎌の一閃（1ターン溜め）

「大鎌振り上げ」の次のターンに発動。防御していなければダメージ後に強制ダウン＋魂状態。防御していれば通常の大ダメージで済む。防御または回復アイテムでの立て直しの読み合いが核となる。

### 拘束からの刈り取り（3ターン猶予）

影の縛りで拘束されたあと、ボスの3ターンの猶予行動を挟んで4ターン目に「拘束刈り取り」が確定発動する。猶予中に脱出できれば回避できる。

## 記念品

- **勝利**: 大鎌の欠片 - 死神の大鎌の刃の欠片。淡く輝き、触れると背筋がひやりとする
- **敗北**: 魂の蝋燭 - 体内の燭台に灯っていた小さな蝋燭。灯すと生気を吸われる気がするが心は安らぐ

## 設定上の注意

- エルナルを死亡・完全消化させる表現は使わない。魂は刈り取られても消えず、脱出すれば体に戻る
- 「死」のモチーフはあくまで可愛らしく、ユーモアと不気味さの同居を保つ
- セラフィムマスコット（善意の救済）との対比として、こちらは「無邪気な収集欲」を軸にする

## メタ・実装メモ

- 内部 ID: `mascot-death` / export 名: `mascotDeathData`
- `explorerLevelRequired: 9`（天空地方の外れ、という位置づけ）
- 一閃の強制ダウンは `onUse` 内で残り HP 分のダメージ＋ `SoulForm` 付与で実装
- 魂の体への帰還は AI 戦略の冒頭で「非・丸呑み中かつ非ダウン」の場合に `SoulForm` を除去して表現
