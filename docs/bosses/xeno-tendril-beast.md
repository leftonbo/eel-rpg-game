# ゼノ触獣 (xeno-tendril-beast)

## 基本情報

- **名前 / 個体名**: ゼノ触獣
- **種族**: 異星触手獣
- **生息地**: 工業地帯外れの隕石孔
- **外見**: 灰白色の外皮と裂けた顎、全身を覆う無数の触手、体内で青白く光る核を持つ
- **体格**: 大型の猛獣ほど。低く伏せて移動し、触手の反動で跳躍する
- **体色**: 灰白色、体内核は青白い発光

## コンセプト

言葉を話さない、触手まみれの異星猛獣。会話や知性の誇示ではなく、唸り声、匂いの追跡、触手の震動で獲物を狩る。中盤以降の海棲触手ボスと差別化するため、宇宙由来の発光粘液と「獣の跳躍」を主軸にする。

## ステータス

- **HP**: 1180
- **攻撃力**: 22
- **解禁レベル**: エクスプローラーLv8

## 設定・背景

工業地帯の外れに落ちた隕石孔から現れた異星生物。知的な言葉は持たず、獲物の匂い、床の振動、体温の変化に反応する。普段は隕石孔の奥に潜み、近づいた生き物を触手で捕まえて体内核の近くへ運ぶ。

## 性格・行動パターン

- 言葉は話せない。鳴き声は「ギィィィ」「シュルルルル」「ヴゥゥゥゥ」など
- 獲物の匂いを覚えると執拗に追う
- 弱った相手を見逃さず、まず拘束してから体内へ運ぼうとする
- 満足すると低く唸り、体内核を青白く明滅させる

## 口調

- なし。意味のある言葉は話さない
- 戦闘ログでは鳴き声、触手音、粘液音、体内の振動で感情を表現する

## 能力・戦術

### 通常状態

- **触手ひっかき**: 標準攻撃。命中率高め
- **牙顎フェイント**: 噛みつくふりから横打ち。低確率で Dizzy
- **発光胞子噴霧**: 発光胞子で VisionImpairment を狙う
- **獲物の匂い刻み**: Weakness を狙いつつ、数ターン拘束を狙いやすくする
- **触手の群れ絡め**: 匂い刻み中に使用頻度が上がる拘束攻撃
- **跳躍姿勢 → 猛獣跳躍**: HP 55% 以下で使う溜め攻撃。次ターンに高威力の跳躍

### 拘束中

- **触手締めつけ**: 拘束中の基本攻撃。少し回復する
- **感覚震動**: Weakness を高確率で付与
- **補食触手**: 活力を吸い上げて大きめに回復

### 食べられ状態

- **体内脈動**: 触手壁が獲物を奥へ押し込む
- **酸性の低鳴り**: 体内の振動と消化液で削り、VisionImpairment を狙う
- **胃内触手まさぐり**: 活力吸収と Dizzy
- **体内核の脈動**: 体内3ターン目以降に一度だけ発生し、Weakness と Dizzy を付与

## 捕食方法

1. 発光胞子や匂い刻みで獲物の感覚と動きを鈍らせる
2. 匂いを覚えた獲物を触手群で拘束する
3. KO + 拘束状態では高確率で丸呑みする
4. 体内核の近くへ運び、触手巣で固定する

## 体内表現

- 体内は青白い核の光と温かな発光粘液で満たされている
- 内壁そのものが触手で、獲物を奥へ押し流す
- 敗北後は核のそばに固定され、8ターンごとに体内粘膜が入れ替わる

## 特殊状態異常

新規状態異常は追加しない。既存の状態異常を組み合わせる。

- **VisionImpairment**: 発光胞子や体内の光で視界を乱す
- **Dizzy**: 牙顎フェイント、胃内触手、体内核の脈動で方向感覚を崩す
- **Weakness**: 匂い刻み、感覚震動、体内核の脈動で力を抜く
- **Slimed**: 敗北後の体内脱皮サイクルで発光粘液に包まれる

## 記念品

- **勝利**: ゼノ触獣の外殻片。灰白色の外皮片で、触手跡のような溝と青白い微光がある
- **敗北**: 核巣の発光粘液。体内核の周囲で採れる温かな粘液で、低い振動を帯びている

## 設定上の注意

- 言葉は話せないため、台詞で意思疎通させない
- 残酷な死亡表現ではなく、体内固定・捕獲・発光粘液で包む表現に留める
- 触手テーマは海のクラーケンと被るため、宇宙由来、猛獣的跳躍、匂い追跡で差別化する
