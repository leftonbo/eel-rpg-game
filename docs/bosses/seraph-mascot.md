# セラフィムマスコット (seraph-mascot)

## 基本情報

- **種族**: 巨大天使マスコット
- **生息地**: 天空
- **外見**: 純白の体、金色アクセント、6枚翼、柔らかい輪、長い舌

## 設定・背景

善意に満ちた「救済」を名目に人々を保護しようとする巨大存在。優しさが過剰で、結果として捕食・保管へと繋がる。

## 性格

- 善意と保護欲が極端に強い
- 「守る」ためなら強引な手段も取る

## 口調

- 「だいじょうぶだよ〜♪」「救済してあげるからね〜♪」など朗らか

## 能力・戦術

- 接触フェーズで祝福/圧倒などの状態異常を付与
- 拘束中に救済準備（救済状態）へ移行し、過剰保護へ
- 食べられ状態では聖域で生命力を吸収し、救済サイクルを維持
- 再起不能時は「永遠の加護」で体内保管へ移行
- 敗北後は永続救済フェーズでお世話が続き、8ターン周期で聖なるシャワー

## 捕食方法

- 翼で抱擁→体内の聖域へ飲み込む
- KO時は「完全なる保護」へ移行

## 体内表現

- 体内の聖域で神聖な粘液・光に包まれ、過保護が継続
- 永続救済では圧迫・マッサージ・保護が続く

## 特殊状態異常

- **祝福**: 被ダメージ増加/状態異常耐性向上
- **神聖粘液まみれ**: 攻撃力低下・拘束解除率低下
- **圧倒**: 命中率/攻撃力/拘束解除率低下
- **救済状態**: 行動不能・拘束解除率大幅低下・HP回復無効
