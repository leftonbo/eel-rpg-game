# Balance, Unlocks, and Trophies

ボスの数値、解禁レベル、記念品を設計するときに読む。

## HP・攻撃力の目安

| 難易度 | HP | 攻撃力 | 用途 |
| --- | ---: | ---: | --- |
| 初級 | 250-450 | 10-24 | 基本システム習得、シンプルなギミック |
| 中級 | 550-800 | 13-22 | 状態異常、拘束、体内行動の組み合わせ |
| 上級 | 970-2600 | 20-32 | 高難易度、複合ギミック、長期戦 |

ギミック特化型は攻撃力を低め、直接火力型は高めにする。実装後は `npm run boss-overview` で既存ボスと比較する。

## 現在の解禁レベル分布

| レベル | テーマ | 既存ボス例 |
| ---: | --- | --- |
| 0 | 初期エリア・基本ボス | 沼のドラゴン、闇のおばけ、機械のクモ |
| 1 | 砂漠 | 運び屋のサソリ |
| 2 | 海 | 海のクラーケン、アクアサーペント、スライムドラゴン |
| 3 | ゲストキャラクター関係 | 夢の淫魔 |
| 4 | ジャングル | 蜜柑ドラゴン |
| 5 | 洞窟・地下世界 | 地底のワーム、舌のドラゴン |
| 6 | 遺跡・廃墟・古城 | クリーンマスター、蝙蝠のヴァンパイア |
| 7 | 氷河・雪山 | ふわふわドラゴン |
| 8 | 工業・施設 | サーマル・アーカイバー |
| 9 | 天空・天使界 | セラフィムマスコット、双面の道化師 |
| 10 | 魔界・冥界 | 魔界の竜 |

テーマが既存エリアと合わない場合は、難易度よりも世界観上の到達順を優先してよい。ただし数値は近いレベル帯のボスと比較する。

## explorerLevelRequired

```typescript
// 最初から利用可能
explorerLevelRequired: 0

// 初級拡張
explorerLevelRequired: 1

// 中級拡張
explorerLevelRequired: 2

// 上級・終盤
explorerLevelRequired: 8
```

未指定時は 0 として扱われるが、新規ボスでは明示する。

## 記念品

各ボスには勝利時と敗北時の記念品を設定できる。

- `victoryTrophy`: ボスの外側から採れるもの。例: 鱗、殻、羽根、外皮、装甲、角、牙。
- `defeatTrophy`: ボスの内側から採れるもの。例: 体液、エッセンス、分泌物、内部で付着した素材。

```typescript
victoryTrophy: {
    name: '記念品名',
    description: 'どのようなアイテムで、ボスのどんな外見的特徴に由来するかを記述。'
},
defeatTrophy: {
    name: '記念品名',
    description: 'ボスの内側で得られた理由と、どんな価値や特徴があるかを記述。'
}
```

記念品名は自動生成も可能だが、ボス固有の特徴を反映した独自設定を推奨する。

## エクスプローラー経験値

記念品と体験したスキルはエクスプローラー経験値に関わる。

1. 初回勝利記念品: `(解禁レベル + 1)^2 * 200`
2. 初回敗北記念品: `(解禁レベル + 1)^2 * 50`
3. スキル体験: `受けたスキル種類数 * 20 * (解禁レベル + 1)`

高レベルボスほど記念品の経験値が大きくなるため、解禁レベルを不用意に上げない。

## バランス確認

実装後に確認すること:

- HP と攻撃力が同レベル帯から大きく外れていない。
- 拘束、睡眠、行動不能など強いギミックを複数持つ場合、攻撃力または命中率を抑えている。
- 体内行動と敗北後行動が長くなりすぎない。
- `victoryTrophy` と `defeatTrophy` がエクスプローラー経験値設計と矛盾しない。
- `npm run boss-overview` で一覧比較している。
